• HostelEase – Hostel Management System
HostelEase is a Hostel Management System developed as a 3rd semester academic project by a team of three members.
The system is designed to streamline hostel operations such as student management, room allocation, and administrative tasks.
I served as the Team Lead for this project, coordinating development, managing task distribution, and contributing to core implementation.

• About This Project
HostelEase was built to simplify and digitize hostel management processes. The system provides an organized way 
to manage student records, room assignments, and administrative workflows.

• Key Features
Student registration and management
Room allocation system
Hostel database management
Admin control panel

• My Role – Team Lead
As the Team Lead, my responsibilities included:
Designing overall system architecture
Dividing tasks among team members
Managing timelines and project progress
Integrating different modules

• Tech Stack
Frontend: HTML, CSS, JavaScript 
Database: Firebase
